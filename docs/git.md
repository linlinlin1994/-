Git的基础命令
=====================


### 新建工程：在你的本地工作目录中进入右键git bash : 执行

    git init

### 添加文件

    git add .

### 注释

    git commit -m '注释写在这里'

### 在github上新建一个仓库记下地址

### 添加工程的源地址

    git  remote add origin (giturl)

### 推送代码到远程仓库

    git push origin master

### 拉取代码到本地

    git pull origin master
